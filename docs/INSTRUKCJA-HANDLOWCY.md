# Instrukcja dla handlowców – Planlux Hale

## Instalacja

1. Pobierz instalator z GitHub Releases (np. `Planlux-Hale-1.0.0.exe`).
2. Uruchom instalator i wybierz folder instalacji.
3. Po instalacji uruchom aplikację z menu Start lub skrótu na pulpicie.

## Aktualizacje

### Automatyczne (zalecane)

- Przy każdym uruchomieniu aplikacja sprawdza, czy jest nowa wersja.
- Gdy pojawi się komunikat **„Nowa wersja dostępna”**:
  1. Kliknij **„Pobierz”**.
  2. Poczekaj na zakończenie pobierania.
  3. Kliknij **„Uruchom ponownie”**.
- Aplikacja zamknie się i zainstaluje aktualizację.

### Ręczne

- Pobierz najnowszy instalator z [GitHub Releases](https://github.com/planlux/planlux-hale/releases).
- Zamknij aplikację i uruchom nowy instalator.
- Zastąpi on poprzednią wersję.

## Wersja aplikacji

- Aktualna wersja jest widoczna w interfejsie (np. w nagłówku lub u dołu ekranu).
- Wersja jest zapisywana w bazie przy logowaniu.

## W razie problemów

- Zgłoś problem administratorowi.
- Zachowaj numer wersji (widoczny w aplikacji).
- Opisz krok po kroku, co się stało.
